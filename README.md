# chatbot-llm-python
Correspondiente al trabajo práctico 3 de Explotación de Datos

Este proyecto es una app de consola en python donde implemento un chatbot centrado en programación.

Proveedor: Google
Modelo LLM utilizado: 'gemini-2.5-flash'
libreria utilizada: 'google-generativeai'

dependencias necesarias a instalar para su uso:
pip install google-generativeai
